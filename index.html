Guía rápida de propiedades CSS
Nombre de la propiedad Sintaxis Descripción de uso Ejemplo de uso Imagen de referencia
color color: <valor>; Define el color del texto. p { color: #c7254e; }
font-size font-size: <longitud|keyword>; Establece el tamaño de la fuente. h1 { font-size: 2rem; }
margin margin: <valores>; Controla el espacio exterior alrededor del elemento. .card { margin: 16px 8px; }
padding padding: <valores>; Controla el espacio interior entre el borde y el contenido. button { padding: 8px 12px; }
border-radius border-radius: <longitud|%>; Redondea las esquinas del borde. img { border-radius: 50%; }
display display: <tipo>; Define el comportamiento de caja (bloque, inline, grid, flex, etc.). nav { display: flex; }
justify-content justify-content: <valor>; Alinea horizontalmente los ítems en un contenedor flex/grid. .row { display:flex; justify-content: space-between; }
grid-template-columns grid-template-columns: <pistas>; Define las columnas de una cuadrícula (Grid). .grid { display:grid; grid-template-columns: 1fr 2fr 1fr; }
background-color background-color: <color>; Define el color de fondo de un elemento. body { background-color: #f0f0f0; }
box-shadow box-shadow: <valores>; Agrega una sombra a un elemento. div { box-shadow: 2px 4px 10px rgba(0,0,0,0.2); }
